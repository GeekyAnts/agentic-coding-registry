## Security: Testing & Validation

Methodologies for security testing and penetration testing.

- **[OWASP WSTG](https://owasp.org/www-project-web-security-testing-guide/):** the Web
  Security Testing Guide — structured test cases across authn / authz, session management,
  input validation, cryptography, business logic, and client-side.
- **[PTES](http://www.pentest-standard.org):** Penetration Testing Execution Standard —
  end-to-end engagement structure.
- **[NIST SP 800-115](https://csrc.nist.gov/pubs/sp/800/115/final):** Technical Guide to
  Information Security Testing and Assessment.

**Apply here:**
- Use the WSTG test cases as the pre-release security-testing checklist.
- Add a regression test for every fixed vulnerability so it cannot silently return.
- Only test systems you are authorized to test; scope and document engagements per PTES.
