# Svelte + Vite — Setup Notes

## Getting started
```
pnpm create vite@latest . --template svelte-ts
pnpm install
pnpm dev
```

## Recommended additions
- ESLint + Prettier (`eslint-plugin-svelte`)
- Vitest + `@testing-library/svelte`
- For routing/SSR, use **SvelteKit** (see the `sveltekit` stack)
