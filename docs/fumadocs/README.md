# Docs (Fumadocs)

[Fumadocs](https://fumadocs.dev) (Next.js) documentation — **starter**.

Fumadocs wires a Next.js App Router + an MDX source pipeline (`source.config.ts`,
`next.config`, route layout). Scaffold the full, current setup with the official
generator, then move your content in:

```
pnpm create fumadocs-app
pnpm dev        # http://localhost:3000
```

- Content lives in `content/docs/` (`.mdx`).
- Fumadocs has a **first-party OpenAPI generator** (`fumadocs-openapi`) for API docs.
- Docs: https://fumadocs.dev
