## MCP Server: Context7

Pulls up-to-date, version-specific documentation and code examples for libraries
straight into context, cutting down wrong-API-signature errors on unfamiliar
packages. Remote HTTP server at `https://mcp.context7.com/mcp`; works without a
key (lower rate limit) — set a `CONTEXT7_API_KEY` (from context7.com) for higher
limits.
